📝 Repository Overview
이 저장소는 오픈소스 소프트웨어(OSS) 강의를 수강하며 학습한 내용을 기록하고 정리하는 공간입니다. Git의 기초부터 고급 기능, 오픈소스 라이선스의 이해까지 실무 중심의 버전 관리 기술을 다룹니다.

📖 주요 학습 내용 (Curriculum)
주차별로 정리된 상세 내용은 OSS 수업내용 정리 폴더에서 확인하실 수 있습니다.

기초 (2~4주차): 버전 관리의 필요성, Git 설정 및 저장소 생성, 커밋(Commit) 및 이력 관리.

관리 (6~8주차): 파일 비교(diff)와 복원(restore), 브랜치(Branch) 생성 및 태그(Tag) 관리, 원격 저장소 동기화.

심화 (10~13주차): 오픈소스 라이선스(GPL, MIT 등)와 stash, 다양한 병합(Merge) 전략과 충돌 해결, rebase 및 reset/revert를 이용한 이력 제어.

## 🛠 주요 기술 및 도구
Version Control: Git (CLI & GUI).

Collaboration: GitHub (Remote Repository, PAT 인증).

Editor: Visual Studio Code (Git 연동 및 확장 프로그램 활용).

Environment: Git Bash (Linux 기초 명령어 활용).
